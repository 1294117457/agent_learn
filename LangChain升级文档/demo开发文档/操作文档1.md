# 操作文档1：idbackend 接入 AI Agent

> 本文档描述后续将 ID-AIDemo Agent 服务接入 idbackend 所需的全部改动。
> **当前阶段无需执行，ID-AIDemo 独立运行中。待 agent 稳定后再执行此文档。**

---

## 一、前提条件

- ID-AIDemo 已部署，运行在 `http://localhost:3001`
- 确认 `GET http://localhost:3001/health` 返回 `{"status":"ok",...}`
- idbackend 所在机器与 ID-AIDemo 在同一主机（或同一内网）

---

## 二、接入设计原则

- **Agent 是额外功能**，agent 宕机不影响原有对话功能
- `ChatController` 优先调 Agent，失败时自动回退到原有 `QianFanService`
- 知识库管理接口独立，仅 ADMIN 角色可访问

---

## 三、application.yml 新增配置

在 `src/main/resources/application.yml`（或 `application-prod.yml`）中追加：

```yaml
ai-agent:
  base-url: http://localhost:3001
  internal-token: id-ai-internal-secret-2024  # 需与 ID-AIDemo .env 中 INTERNAL_TOKEN 一致
  timeout:
    connect: 2000    # 连接超时 2秒（agent 不在线时快速失败）
    read: 30000      # 读取超时 30秒（Qwen3 生成可能较慢）
```

## 四、新增文件：AIAgentService.java

**路径**：`src/main/java/com/zch/idbackend/service/functionService/AIAgentService.java`

```java
package com.zch.idbackend.service.functionService;

import lombok.extern.slf4j.Slf4j;
import org.springframework.beans.factory.annotation.Value;
import org.springframework.http.*;
import org.springframework.stereotype.Service;
import org.springframework.web.client.RestTemplate;
import org.springframework.boot.web.client.RestTemplateBuilder;

import java.time.Duration;
import java.util.Map;

@Slf4j
@Service
public class AIAgentService {

    private final RestTemplate restTemplate;

    @Value("${ai-agent.base-url}")
    private String baseUrl;

    @Value("${ai-agent.internal-token}")
    private String internalToken;

    public AIAgentService(
        @Value("${ai-agent.timeout.connect:2000}") int connectTimeout,
        @Value("${ai-agent.timeout.read:30000}") int readTimeout
    ) {
        this.restTemplate = new RestTemplateBuilder()
            .connectTimeout(Duration.ofMillis(connectTimeout))
            .readTimeout(Duration.ofMillis(readTimeout))
            .build();
    }

    /**
     * RAG 对话，agent 不可用时返回 null（由调用方降级）
     */
    public String chat(String message, Long userId) {
        try {
            HttpHeaders headers = new HttpHeaders();
            headers.setContentType(MediaType.APPLICATION_JSON);
            headers.set("X-Internal-Token", internalToken);

            Map<String, Object> body = Map.of(
                "sessionId", "user_" + userId,
                "message", message
            );

            ResponseEntity<Map> resp = restTemplate.postForEntity(
                baseUrl + "/chat/send",
                new HttpEntity<>(body, headers),
                Map.class
            );

            if (resp.getStatusCode().is2xxSuccessful() && resp.getBody() != null) {
                Map<?, ?> data = (Map<?, ?>) resp.getBody().get("data");
                if (data != null) {
                    return (String) data.get("reply");
                }
            }
            return null;
        } catch (Exception e) {
            log.warn("[AIAgent] chat 不可用，将降级到 QianFan: {}", e.getMessage());
            return null;
        }
    }

    /**
     * 清除 agent 侧会话历史（失败不影响主流程）
     */
    public void clearConversation(Long userId) {
        try {
            HttpHeaders headers = new HttpHeaders();
            headers.setContentType(MediaType.APPLICATION_JSON);
            headers.set("X-Internal-Token", internalToken);

            restTemplate.postForEntity(
                baseUrl + "/chat/clear",
                new HttpEntity<>(Map.of("sessionId", "user_" + userId), headers),
                Map.class
            );
        } catch (Exception e) {
            log.warn("[AIAgent] clearConversation 失败（忽略）: {}", e.getMessage());
        }
    }

    /**
     * 健康探测（可选，供监控使用）
     */
    public boolean isAvailable() {
        try {
            ResponseEntity<Map> resp = restTemplate.getForEntity(baseUrl + "/health", Map.class);
            return resp.getStatusCode().is2xxSuccessful();
        } catch (Exception e) {
            return false;
        }
    }
}
```

## 五、修改文件：ChatController.java

**路径**：`src/main/java/com/zch/idbackend/controller/businessController/ChatController.java`

修改点：注入 `AIAgentService`，优先调用 agent，失败回退 QianFan。

```java
@RestController
@RequestMapping("/api/chat")
@RequiredArgsConstructor
public class ChatController {

    private final QianFanService qianFanService;
    private final AIAgentService aiAgentService;   // ← 新增注入

    @PostMapping("/send")
    public ResultVo chat(@RequestBody ChatRequestDTO request, HttpServletRequest httpRequest) {
        if (request.getMessage() == null || request.getMessage().trim().isEmpty()) {
            return ResultVo.error(400, "消息不能为空");
        }

        Integer userIdInt = (Integer) httpRequest.getAttribute("userId");

        String reply = null;

        // 优先尝试 Agent（RAG）
        if (userIdInt != null) {
            reply = aiAgentService.chat(request.getMessage(), userIdInt.longValue());
        }

        // Agent 不可用时降级到 QianFan
        if (reply == null) {
            if (userIdInt != null) {
                reply = qianFanService.chat(request.getMessage(), userIdInt.longValue());
            } else {
                reply = qianFanService.chat(request.getMessage());
            }
        }

        return ResultVo.success("成功", reply);
    }

    @PostMapping("/clear")
    public ResultVo clearConversation(HttpServletRequest httpRequest) {
        Integer userIdInt = (Integer) httpRequest.getAttribute("userId");
        if (userIdInt == null) {
            return ResultVo.error(401, "用户未登录");
        }
        Long userId = userIdInt.longValue();
        aiAgentService.clearConversation(userId);    // ← 同步清 agent 侧历史
        qianFanService.clearConversation(userId);    // ← 保留原逻辑
        return ResultVo.success("对话已清除");
    }
}
```

## 六、新增文件：KnowledgeController.java（知识库管理，ADMIN 专用）

**路径**：`src/main/java/com/zch/idbackend/controller/businessController/KnowledgeController.java`

```java
package com.zch.idbackend.controller.businessController;

import com.zch.idbackend.controller.dto.ResultVo;
import lombok.RequiredArgsConstructor;
import lombok.extern.slf4j.Slf4j;
import org.springframework.beans.factory.annotation.Value;
import org.springframework.core.io.ByteArrayResource;
import org.springframework.http.*;
import org.springframework.util.LinkedMultiValueMap;
import org.springframework.util.MultiValueMap;
import org.springframework.web.bind.annotation.*;
import org.springframework.web.client.RestTemplate;
import org.springframework.web.multipart.MultipartFile;

import java.util.Map;

/**
 * 知识库管理接口（仅 ADMIN 角色可访问，由 JWT 拦截器保证）
 */
@Slf4j
@RestController
@RequestMapping("/api/knowledge")
@RequiredArgsConstructor
public class KnowledgeController {

    private final RestTemplate restTemplate;   // 注入通用 RestTemplate

    @Value("${ai-agent.base-url}")
    private String agentBaseUrl;

    @Value("${ai-agent.internal-token}")
    private String internalToken;

    /** 查询知识库文件列表 */
    @GetMapping("/list")
    public ResultVo list() {
        try {
            HttpHeaders headers = new HttpHeaders();
            headers.set("X-Internal-Token", internalToken);
            ResponseEntity<Map> resp = restTemplate.exchange(
                agentBaseUrl + "/knowledge/list",
                HttpMethod.GET,
                new HttpEntity<>(headers),
                Map.class
            );
            return ResultVo.success("成功", resp.getBody() != null ? resp.getBody().get("data") : null);
        } catch (Exception e) {
            log.error("[Knowledge] list 失败", e);
            return ResultVo.error(503, "AI服务暂时不可用");
        }
    }

    /** 上传文档到知识库 */
    @PostMapping("/upload")
    public ResultVo upload(@RequestParam("file") MultipartFile file) {
        try {
            HttpHeaders headers = new HttpHeaders();
            headers.setContentType(MediaType.MULTIPART_FORM_DATA);
            headers.set("X-Internal-Token", internalToken);

            ByteArrayResource fileResource = new ByteArrayResource(file.getBytes()) {
                @Override public String getFilename() { return file.getOriginalFilename(); }
            };

            MultiValueMap<String, Object> body = new LinkedMultiValueMap<>();
            body.add("file", fileResource);

            ResponseEntity<Map> resp = restTemplate.postForEntity(
                agentBaseUrl + "/knowledge/upload",
                new HttpEntity<>(body, headers),
                Map.class
            );
            return ResultVo.success("上传成功", resp.getBody() != null ? resp.getBody().get("data") : null);
        } catch (Exception e) {
            log.error("[Knowledge] upload 失败", e);
            return ResultVo.error(503, "AI服务暂时不可用");
        }
    }

    /** 删除知识库文件 */
    @DeleteMapping("/{sourceFile}")
    public ResultVo delete(@PathVariable String sourceFile) {
        try {
            HttpHeaders headers = new HttpHeaders();
            headers.set("X-Internal-Token", internalToken);
            restTemplate.exchange(
                agentBaseUrl + "/knowledge/" + sourceFile,
                HttpMethod.DELETE,
                new HttpEntity<>(headers),
                Map.class
            );
            return ResultVo.success("删除成功");
        } catch (Exception e) {
            log.error("[Knowledge] delete 失败", e);
            return ResultVo.error(503, "AI服务暂时不可用");
        }
    }
}
```

## 七、可选：给 ID-AIDemo 添加 Token 校验

当前 ID-AIDemo 未验证 `X-Internal-Token`（所有接口公开）。若需要安全加固，在 `src/index.ts` 中加入中间件：

```typescript
// 在 app.use('/chat', ...) 之前加入：
app.use(['/chat', '/knowledge'], (req, res, next) => {
  const token = req.headers['x-internal-token']
  if (token !== process.env.INTERNAL_TOKEN) {
    res.status(403).json({ code: 403, msg: 'Forbidden' })
    return
  }
  next()
})
```

对外开放 `/health`（用于探测），不做 token 校验。

## 八、前端 sessionId 说明

现有前端的 `apiAIagent.ts` 调用 `POST /api/chat/send` 只传 `{ message }`，不传 `sessionId`。

idbackend 接收到请求后，从 JWT 中取 `userId`，在 `AIAgentService.chat()` 内部构造 `sessionId = "user_" + userId`，无需前端修改。

## 十、部署注意事项

1. ID-AIDemo 先于 idbackend 启动（或 idbackend 容忍 agent 不在线）
2. ID-AIDemo 作为后台进程运行：`pnpm start &` 或使用 PM2 管理
3. `.env` 中的 `INTERNAL_TOKEN` 与 `application.yml` 中的值保持一致
4. 生产环境建议将 ID-AIDemo 监听 `127.0.0.1`（仅本机）而不是 `0.0.0.0`
